# Gaska.dev - Personal Portfolio

Gaska.dev adalah website portofolio personal milik Farrel Gaska Haryadi. Website ini dibangun untuk menampilkan profil, keahlian, serta project-project yang pernah dikerjakan dengan antarmuka yang modern.

Project ini dikembangkan menggunakan React dan Vite untuk performa yang cepat, serta Tailwind CSS untuk styling yang rapi dan responsif, sehingga sangat cocok dijadikan sebagai representasi digital profesional.

## Demo

Live Demo:

https://farrelgaska.github.io/Portofolio-Gaska.dev/

## Fitur Utama

- Menampilkan informasi profil dan perkenalan diri
- Showcase daftar project dan pengalaman
- Desain antarmuka modern dan *clean* menggunakan Tailwind CSS
- Mendukung fitur *dark mode* (berdasarkan konfigurasi Tailwind)
- Desain sepenuhnya responsif untuk berbagai ukuran layar (mobile, tablet, desktop)
- *Fast loading* berkat sistem *build* dari Vite

## Teknologi yang Digunakan

- React.js
- Vite
- Tailwind CSS
- HTML5
- CSS3
- JavaScript (ES6+)
- Google Fonts (Anton & Manrope)
